Azure 기본 사항 파트 1: 핵심 Azure 개념에 관해 설명하기

  • 1시간 23분 남음
  • 학습 경로
  • 0/3개 모듈 완료
초급
관리자
솔루션 아키텍처
개발자
학생
Azure
Azure Resource Manager
Functions

클라우드는 처음이신가요? Azure 기본 사항은 6부 시리즈로 클라우드의 기본 개념을 설명하고 다양한 Azure 서비스의 간소화된 개요를 제공합니다. 첫 번째 서비스를 추가 비용 없이 배포할 수 있는 실습도 안내합니다.

이 학습 경로를 완료하고 나면 다음을 수행할 수 있습니다.

  • Azure 클라우드 컴퓨팅의 장점과 이를 통해 시간과 비용을 절감하는 방법 이해
  • 고가용성, 확장성, 탄력성, 민첩성 및 재해 복구와 같은 클라우드 개념에 관해 설명하기
  • 구독, 관리 그룹, 리소스 및 리소스 그룹과 같은 핵심 Azure 아키텍처 구성 요소에 관해 설명하기
  • Azure 지역, 지역 쌍 및 가용성 영역과 같은 지리적 배포 개념 요약하기

이 시리즈의 학습 경로는 다음과 같습니다.

1부: 핵심 Azure 개념에 관해 설명하기

2부: 핵심 Azure 서비스에 관해 설명하기

3부: Azure의 핵심 솔루션 및 관리 도구에 관해 설명하기

4부: 일반 보안 및 네트워크 보안 기능에 관해 설명하기

5부: ID, 거버넌스, 개인 정보 보호 및 규정 준수 기능에 관해 설명하기

6부: Azure Cost Management 및 서비스 수준 계약에 관해 설명하기

시험 AZ-900: Azure 기본 사항을 준비하는 경우 시리즈의 모든 학습 경로를 완료합니다.

필수 조건

  • 기본적인 컴퓨팅 개념과 용어를 잘 알고 있어야 합니다.
  • 클라우드 컴퓨팅에 관한 이해가 도움이 되지만 꼭 필요한 것은 아닙니다.
